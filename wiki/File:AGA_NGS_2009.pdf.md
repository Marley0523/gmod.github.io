# <span id="Seeing_the_forest_and_the_trees:_visualizing_next_generation_sequence_data" class="mw-headline">Seeing the forest and the trees: visualizing next generation sequence data</span>

Presentation by [Dave Clements](User:Clements "User:Clements") at
*<a href="http://www.regonline.com/Nextgeneration" class="external text"
rel="nofollow">Next Generation Genome Analysis in Non-Model Organisms An
AGA Special Event</a>*, June 13, 2009, at the University of Connecticut.

Discusses using [GBrowse](GBrowse.1 "GBrowse") and SAMtools to visualize
<a href="Next_generation_sequencing" class="mw-redirect"
title="Next generation sequencing">Next Generation Sequence Data</a>.
